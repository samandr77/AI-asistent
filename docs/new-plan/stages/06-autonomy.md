# Stage 06 - Autonomy And Agents

## Goal

Add controlled autonomy: scenarios, triggers, routines, proactive suggestions, agent execution, and full transparency.

## Features

- Automation scenarios with trigger, condition, action, permission level, enabled state.
- Trigger types: time, event, context, metric threshold.
- Permission levels: suggest only, require confirmation, allowed autonomous action.
- Full audit log for suggestions, approvals, rejections, and executed actions.
- Routine templates: morning, evening, weekly, medication, hydration, budget review, focus block.
- Adaptive reminders based on completion history and user feedback.
- Agent loop: understand, plan, ask permission, execute, verify, report.
- Proactive alerts for deadlines, overload, low recovery, budget overspend, subscriptions, missed routines.
- Weekly and monthly autonomous reports.
- Communication automation contracts with reply drafts, classification, style memory, and channel restrictions.

## Acceptance Criteria

- No high-impact action executes without the right permission level.
- User can disable any scenario.
- Audit log shows what happened, why, and which data was used.
