## 3. Цели обработки

- Предоставление функциональности приложения (разбор дампов в задачи, хранение задач/целей/рефлексий)
- Аутентификация и защита аккаунтов
- Диагностика сбоев и улучшение качества
- Обработка подписок через App Store / Google Play
- Вход, напоминания и Premium-покупки внутри Telegram Mini App / bot

## 4. Правовое основание

- Исполнение договора (предоставление сервиса) — GDPR ст. 6(1)(b)
- Законный интерес (безопасность, диагностика) — GDPR ст. 6(1)(f)
- Согласие (там, где применимо) — GDPR ст. 6(1)(a)

## 5. Срок хранения

- Данные активного аккаунта — до удаления аккаунта пользователем.
- После удаления аккаунта — данные окончательно стираются **через 30 дней**. В течение этих 30 дней возможен запрос на восстановление через support@second-brain.app.
- Диагностика (Sentry) — 90 дней, затем автоматическое удаление.

## 7. Права пользователя (GDPR)

Пользователь имеет право:
- Получить копию своих данных (ст. 15) — запрос на support@second-brain.app.
- Исправить неточности (ст. 16) — через интерфейс приложения или поддержку.
- Удалить данные (ст. 17) — через Профиль → "Удалить аккаунт". Также возможен запрос через поддержку.
- Ограничить обработку (ст. 18) — запрос через поддержку.
- Перенести данные в другой сервис (ст. 20) — через экспорт, отправляемый по запросу.
- Подать жалобу в надзорный орган (ст. 77).

## 8. Безопасность

- Шифрование TLS 1.2+ для всей передачи данных.
- Row-Level Security в Postgres: никто, кроме владельца, не может прочитать записи.

## 9. Telegram Mini App и web storage

Telegram Mini App использует защищённые launch data Telegram (`initData`) только для проверки входа на backend. В web-клиенте может храниться локальный session token и очередь неотправленных текстовых дампов, чтобы приложение продолжало работать при кратком отсутствии сети. Cookies для рекламного трекинга не используются.
