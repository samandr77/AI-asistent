# Stage 03 - Tasks And Productivity

## Goal

Implement a complete GTD-style task system connected to goals, time, focus, analytics, and AI planning.

## Features

- Quick capture from text, voice, Telegram bot, browser/share, email/manual placeholder.
- NLP task parsing for title, date, time, reminder, contact, priority, project, and context.
- Smart inbox with process actions: do now, schedule, delegate, delete, convert to project.
- Eisenhower matrix and Big Three daily planning.
- Time blocking with calendar slot, estimated duration, actual duration, deep-work flag, overload warning.
- Pomodoro/focus mode linked to tasks and stats.
- Projects with subtasks, checklists, dependencies, templates, progress, archive.
- Recurring tasks and habits with rollover, chains, frequency rules, and context conditions.
- Contexts, tags, smart filters, saved filters, geolocation/context placeholder.
- Collaboration contracts for assignee, comments, files, statuses, mentions.
- Productivity analytics and weekly report.

## Acceptance Criteria

- Capture works without opening complex forms.
- Inbox processing never loses original user text.
- Recurring tasks do not duplicate uncontrollably.
- Analytics compare estimates with real completion history.
