# Stage 02 - Control Center

## Goal

Create the main hub that connects daily work to goals, health, finance, events, energy, and AI recommendations.

## Features

- Main dashboard with top-3 priorities, active goals, health signals, finance signals, upcoming tasks, and events.
- OKR hierarchy: life goals, yearly OKR, quarterly OKR, weekly sprint, daily tasks.
- Strategic map: mission, values, life areas, SWOT, priority weights.
- KPI tracker with 5-10 chosen metrics, trends, correlations, and warnings.
- Morning briefing and energy-aware day plan.
- Daily review and weekly review.
- Knowledge-base entry points for PARA and graph/RAG search.

## Acceptance Criteria

- Dashboard can render with partial data from any module.
- User can override AI priorities and KPI selection.
- Daily and weekly reviews can be saved, edited, and used by AI memory.
- AI recommendations explain which data they used.
